Identifying Diseases in Chest X-Rays & COVID-19 Detection

Project Information

  • Category: Image Analysis
  • Chapter: Omdena Myanmar Chapter
  • Project duration: 07/23 - ongoing
  • Project URL:
    Project's Official Page

This project addresses the dual challenges of inadequate diagnostic tools for chest diseases, including COVID-19, and a shortage of healthcare professionals in Myanmar. By leveraging AI, we aimed to enhance chest X-ray analysis accuracy and speed.
